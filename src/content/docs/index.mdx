เอกสารชุดนี้พาคุณตั้งค่า `Revit 2026/2027 plugin` ตั้งแต่เตรียมเครื่องจนถึงการทดสอบและ debug ใน `Visual Studio`

## สิ่งที่คุณจะได้เมื่อทำจบ

- โปรเจ็กต์ `net8.0-windows` (หรือ `net10.0-windows`) และ `x64` ที่ reference `RevitAPI.dll` และ `RevitAPIUI.dll` ถูกต้อง
- plugin แบบ `IExternalCommand` สำหรับ command-first และ `IExternalApplication` สำหรับสร้าง Ribbon/UI
- workflow สำหรับ copy `DLL` ไป deploy folder คงที่ และ copy `.addin` ไป `ProgramData`
- debug profile ที่กด `F5` แล้วเปิด `Revit.exe` พร้อม attach debugger ได้
